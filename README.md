# figurines
Dispositif de lecture et de dérive de texte brut. Chaque ligne devient une figurine placée sur une grille de 12 colonnes, soumise à mémoire locale, dérive, interaction utilisateur et mémoire globale. Le texte se reconfigure en continu, produisant des formes instables où le sens émerge des relations plutôt que de la linéarité.
